# Smart Expense Tracker

A modern expense tracking application built using HTML, CSS, and JavaScript.

## Features

- Add income and expenses
- Categorize transactions
- Automatic balance calculation
- Local storage persistence
- Delete transactions
- Responsive design

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## Installation

1. Clone repository

```bash
git clone https://github.com/Diginewgen/smart-expense-tracker.git
```

2. Open `index.html` in your browser
